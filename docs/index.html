About J.S.W.A.T

What is J.S.W.A.T?

What is the "J.S" for? "JavaScript?" Could be. How about "Just Simple," or "Just Smart"? J.S.W.A.T stands for JavaScript Web Application Template. A simple web application template built ontop of Node.js and the EJS templating language, allowing you to generate HTML markup with plain JavaScript. No need for constantly creating development projects with clicks and clacks and all the nitty gritty! We know how to organize and simplify things . No reinvention of the wheel and control-flow. It's just plain JavaScript with a bit of fun!

Responsive Execution

We know how fast the Chrome V8 engine and the other JavaScript runtimes have become. With EJS, caching is intermediate with JS functions for quick-paced execution and build time.

A Passion for JS

We love JavaScript. It's a totally awesome, user-friendly programming language. Most templating languages and engines grow out to be complex and difficult to understand. Why not cut out the middle-man, and use J.S.W.A.T!

Speedy Development Time

Don't waste time and efforts figuring out alien new syntax because of 'trends'. Learning how to pre-process your data and code so it will actually render right isn't easy when you're starting out with an empty folder and no direction.

Simple-Simon Syntax

Plain JavaScript code in simple, straightforward EJS scriptlet tags. Just write JavaScript that emits the HTML you want, and get the job done! EJS supports both ES5 & ES6 syntax.

Easy-Peasy Debugging

Debugging EJS: Debugging has never been so efficient and simple, your errors are displayed in plain JavaScript exceptions right inside your browser, with template line-numbers indicating the traceback region for your convenience.

J.S.W.A.T-X NEW!
a pre-built, pre-designed component library

Using the best CSS3 libraries out there, Bootstrap & W3S have made frontend design even simpler and cleaner. We put together a clean, easy library that allows you to render a component with a single, simple "-Include" statement from EJS. No more messy classes for you!

Supports Backend & Frontend Web Projects

An integrated JavaScript eco-system. No need to set up a backend. The template has it all done for you! Integrating with API's & other backend services will be a drift in the wind.

Open-Source

J.S.W.A.T is Free to use, modify and distribute. A lifetime solution All-In-One and ever-evolving.

Features

Key features of J.S.W.A.T.

Quick-paced build, compilation and render time

Built & designed with in the Republic of South Africa.

Efficient on power & memory consumption and CPU usage.

Use "[? ?]" instead of "<% %>"
